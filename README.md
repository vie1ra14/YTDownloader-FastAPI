# YTDownloader-FastAPI



## About


YTDownloader is a simple YouTube audio downloader built with FastAPI.

Just paste a YouTube video URL and the application will download the audio as an MP3 file.

The project was created as a lightweight backend-focused implementation using FastAPI and yt-dlp

## Technologies & Libraries

- **FastAPI** - backend API framework

- **yt_dlp** - ddownloading and extracting audio from YouTube videos

- **urllib.parse** - URL parsing and validation


## How to Use

1. Paste the YouTube video URL in the "YouTube URL" field.

2. Click Start Download.

3. The application will process the video and download the audio as an MP3 file.
<br>
<br>

You can also test the API directly through the FastAPI Swagger UI.
